## 如何选择

| 产品 | 更适合谁 | 典型场景 |
| --- | --- | --- |
| AOS Desktop | 希望通过图形界面完成扫描、研判和处置的安全团队、研发团队和产品团队 | 本地 Agent 生态盘点、Skill 风险分析、隔离处置、运行时活动监控 |
| AOS CLI | 希望把安全检查嵌入终端、自动化流程或 CI 的开发者和平台团队 | 命令行扫描、代码库审计、JSON 输出、脚本化集成 |

## 首次使用建议

1. 如果你想先从产品全貌和可视化界面开始，先看 [AOS Desktop](/docs/aos-desktop/)。
2. 如果你已经在终端里管理 Agent 工作流，直接看 [AOS CLI](/docs/aos-cli/)。
3. 如果你的重点是运行时行为观测，继续看 [AOS Desktop / 活动监控](/docs/aos-desktop/activity-monitor/)。

## 你会在文档里看到什么

- 产品定位与能力边界：帮助你快速判断什么时候该用桌面端，什么时候该用 CLI。
- 快速开始：用最短路径完成第一次扫描和结果查看。
- 专题说明：补充运行时活动监控、命令选项、输出方式等进阶内容。
